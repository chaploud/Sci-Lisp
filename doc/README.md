# Sci-Lisp Documents

## Prerequests

### Linux

- g++
- CMake

### Windows

Unsupported yet.

### Mac

Unsupported yet.

## Installation

```bash
# Clone Repository
git clone https://github.com/chaploud/Sci-Lisp.git

# build scilisp binary using CMake
cd Sci-Lisp
./build.sh  # => biuld/scilisp
```

## Command Options

```bash
scilisp  # launch REPL
scilisp xxx.lisp  # run as script
scilisp -c xxx.lisp  # compile code
scllisp -l xxx.lisp  #lint code
```
